# 📊 Proyectos de Ciencias Actuariales

**Eduardo Carrillo Reyna** · Facultad de Contaduría y Administración · Universidad de Querétaro

## 📁 Proyectos Destacados

### 🔢 Matrices de Leslie
Modelos matriciales para proyección de poblaciones con estructura por edad. Implementación en Python con análisis de eigenvalores y tasas de crecimiento.

### 📈 Simulaciones Monte Carlo
Simulaciones estocásticas aplicadas a valoración de instrumentos financieros y análisis de riesgo en portafolios de inversión.

### 💼 Portafolio de Inversión
Construcción de portafolios con frontera eficiente, análisis de ETFs y optimización media-varianza (modelo de Markowitz).

### 🏦 Sistema de Pensiones
Análisis actuarial del sistema de pensiones mexicano (IMSS/Afores), proyecciones financieras y modelos de beneficios.
